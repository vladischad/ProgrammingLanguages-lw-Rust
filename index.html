Rust Programming Language

By: Vlad Maliutin, Georgia Henson, Tiernan Benner, Evan Lauer

Introduction and History

Origins: A Passion Project (2006)

Rust began as a personal project by Graydon Hoare in 2006. Working independently, he sought to create a language that combined the control and performance of C++ with better safety guarantees. His goal was to address common issues like memory errors and concurrency bugs without relying on a garbage collector, which can introduce runtime overhead.

Mozilla's Sponsorship (2009)

Recognizing the potential of Rust, Mozilla began sponsoring the project in 2009. Rust moved from a personal experiment to an officially backed initiative within Mozilla Research. Over the next few years, a team of contributors helped refine Rust's concepts, and Mozilla positioned it as a core tool to build safer and faster web technologies.

Major Advancements

  • 2010: First public release of Rust's early prototype.
  • 2012: Servo, an experimental browser engine written in Rust, was started by Mozilla, showcasing Rust's concurrency and safety features.
  • 2015: Release of Rust 1.0, the first stable version of the language — marking Rust’s commitment to long-term stability for its users.
  • 2018: Major improvements with Rust 2018 Edition, adding features like async/await syntax and a smoother developer experience.
  • 2021: Formation of the independent Rust Foundation to steward the language’s future development, funded by companies like AWS, Google, Microsoft, and Mozilla.

Key Features

  • Memory Safety Without Garbage Collection: Rust uses a unique ownership system that enforces safety at compile-time without needing a garbage collector.
  • Fearless Concurrency: The language design makes writing multi-threaded programs easier and safer, avoiding data races.
  • Performance: Rust programs are typically as fast as C and C++ programs due to low-level control over memory and computation.
  • Rich Type System and Pattern Matching: These features make programs more expressive and robust.
  • Friendly Tooling: Tools like Cargo (package manager and build system) and Rust Analyzer make development smooth and accessible.

How Rust is Used Today

  • Systems Programming: Used in operating system kernels, file systems, and browser engines (e.g., Servo and parts of Firefox).
  • Web Development: Frameworks like Rocket and Actix Web allow for building high-performance web servers and APIs.
  • Cloud and Embedded Systems: Companies like Amazon use Rust for cloud services, and it’s gaining adoption in embedded systems for its safety guarantees.
  • Blockchain Development: Many blockchain projects, including parts of Solana and Polkadot, use Rust for their core infrastructure.
  • Tooling and Developer Utilities: Popular tools like ripgrep (a search tool) and exa (a modern `ls` replacement) are built with Rust.

Growing Popularity

Since 2016, Rust has consistently been voted the "Most Loved Programming Language" in Stack Overflow’s annual Developer Surveys. Developers praise its focus on reliability, performance, and developer experience, despite its steep initial learning curve.

Community and Open Governance

Rust is now managed by the Rust Foundation, ensuring that it remains a community-driven and openly governed project. Contributions come from individuals and companies worldwide, making Rust one of the most collaboratively developed modern languages.

Fibonacci Generator

fn fib(n: u32) -> u32 {
              match n {
                  0 => 0,
                  1 => 1,
                  _ => fib(n - 1) + fib(n - 2),
              }
            }
fn main() {
              let n = 10;
              println!("fib({}) = {}", n, fib(n));
          }
Build/Run: 
rustc main.rs
          ./main

Advanced Example

Calculator

fn add(a: i32, b: i32) -> i32 {
              a + b
            }

            fn subtract(a: i32, b: i32) -> i32 {
              a - b
            }

            fn multiply(a: i32, b: i32) -> i32 {
              a * b
            }

            fn divide(a: i32, b: i32) -> f64 {
              a as f64 / b as f64
            }
            
fn main() {
              let a = 10;
              let b = 5;
              println!("{} + {} = {}", a, b, add(a, b));
              println!("{} - {} = {}", a, b, subtract(a, b));
              println!("{} * {} = {}", a, b, multiply(a, b));
              println!("{} / {} = {}", a, b, divide(a, b));
            }

Build/Run: 
rustc main.rs
          ./main

Use of Generics

trait Shape {
                    fn area(&self) -> f64;
                }
                
struct Circle {
                    radius: f64,
                }

                
struct Rectangle {
                    width: f64,
                    height: f64,
                }

                 
impl Shape for Circle {
                    fn area(&self) -> f64 {
                        std::f64::consts::PI * self.radius * self.radius
                    }
                }

                
impl Shape for Rectangle {
                    fn area(&self) -> f64 {
                        self.width * self.height
                    }
                }

                
                
fn print_area(shape: &T) {
                    println!("Area: {}", shape.area());
                }

                
            
fn main() {
                    let c = Circle { radius: 5.0 };
                    let r = Rectangle { width: 4.0, height: 6.0 };
                
                    print_area(&c);
                    print_area(&r);
                
}
